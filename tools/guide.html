Colors

The colors can be chosen in 2 ways, the color picker (2 buttons to the right of the Clone button) or the 6 HSL inputs. 1st 3 are the main color, other 3 are the outline color.

The clone button duplicates the colors and the "+" button adds a new one with default values.

To choose the color you want to use to draw on the card you need to click on one of the two squear to the right of the HSL inputs. They also serve as a preview of the color so you know which one is which.

Card data

The card has 5 inputs that contain data about the tiles.
1) Tiles count in a row (left to right). Default is 12
2) Tiles count in a column (top to bottom). Default is 9
3) the ammount of pixels the outline of a tile has. Default is 2
4) the width of the tile. Default is 46
5) the height of the tile. Default is 46

Text - the easy part

The text is an important part of the card. It has lots of different thing that can be changed.

The buttons at the top:
Draw: Draws the text if the button green.
Hide: Hides the iformation about the text.
Clone: Clones the text.
Move/Stop: Sets to move/static mode. Makes you able to move the text with the mouse. The coordinates that were in "tl", will be changed in "tl". Others will be changed in pixels.

font: Any font that is builtin or is on Google fonts.
text: The text that is displayed in the card. To make a new line you need to press the "enter" key.
color: The color of the text.
alpha: The alpha of the color.
outline: The color of the outline.
alpha: The alpha of the outline.
shadow color: The color of the shadow. Note, the shadow strength depends on the alpha of the outline or the color.
x and y: The position of the text on the X. It may be "center", a number (in pixels) or with a "tl" at the end (e.g. 5tl). the tl means "tile".
size: The size of the font.
outline size: The size of the outline.
shadow size: The size of the shadow.
line height: The new line distance from the line above.
bold: Makes the text bold.
italic: Makes the text italic.
align: The text align. Left Center or Right.

Text - the hard part

The hard part is text editing. As you can see in the picture, you can use different colors, outlines e.t.c. in 1 text.
You use them by putting them in to a <key=value>.
The color ones may accept a color like #123456, rgb(255,255,0), hsl(100, 50%, 50%).
The size and x,y ones are a number like 1, 6.65, -24 (only x and y can "actually" be negative). X and Y are calculated from the place they were called. <x=10> will move the text 10 pixels to the right.
Bold and italic dont neev a value. its just <b> and <i>.

Text - the very hard part (Variables)

You can create a variable. But what does it do?
A variable is a thing that may shorten your text.
To use them you do <var=varName;Argument0;Argument1...>.
What?
Lets say you want a variable that will act as a custom color.
You will do key: myColor, value: <c=red>. This will make a variable that does the same as <c=red>
So how do i use them?
In the text you will type <var=myColor;>. Done!
Ok.. So what would i even use it?
Lets say you have 5 parts where you need a different color. if you do it with multiple text, editing it will be a pain. If you do it by <c=red>, Imagine changing it. You will be suppost to find all 5 sport and change them multiple times untill you find the color you need.
But if you do it with a var, all you need to do is to change the vars value!
Ok. So what are the arg0, arg1 etc?
In the vars value you can add a {0}, {1} etc..
the {0} will be changed by the 1st argument, {1} by the second and so on.
